## Project introduction

**유튜브를 보다가 알고리즘에 따라서 추천되는 영상들을 이유없이 계속 시청한 경험이 한번쯤은 있을 겁니다.**

**시청하는 동안 휴식을 가진다 생각하지만 막상 그렇게 시간을 가지고 나면 자극적인 영상들이 머리속에 남아**

**휴식한 것 같지 않은 느낌이 들 때가 있죠.**
